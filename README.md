# Ruger Mark IV 22/45 Bolt Ears Model

1) BoltEars.SLDPRT
2) BoltEars.STL

These are the model of the back of the bolt such that they can be used to model a bolt racker/charger

# Ruger Mark IV 22/45 Bolt Charger

1) RackerV2.SLDPRT
2) V2_Racker_ASM.SLDASM
3) RackerV2.STL

These are the model bolt charger that rotates onto the ears of the bolt. The lever is angled up away from the grip for comfort and ease of function. Hardware for completeness can be found in Assembly file. (McMaster Carr PNs: 92505A242 and 91333A130)

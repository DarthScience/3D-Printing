# Ruger Mark IV 22/45 

##   1) Bolt Ears
  This is the model of the back of the bolt such that they can be used to model a bolt racker/charger

      A) BoltEars.SLDPRT
      B) BoltEars.STL

##    2) Bolt Charger
  These are the models for a bolt charger that rotates onto the ears of the bolt. The pull lever arm is angled up away from the grip for  comfort and ease of function. Hardware for completeness can be found in Assembly file. (McMaster Carr PNs: 92505A242 and 91333A130) 

      A) RackerV2.SLDPRT
      B) V2_Racker_ASM.SLDASM
      C) RackerV2.STL
